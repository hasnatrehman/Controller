Readme - Flood & Rain Puddles Shaders
_________________________

Support Unity Versions :

- URP : 2022 LTS 
(For 2019 to 2021 Unity version, unpack the URP - Wet Flood&Rain Puddles Shaders_v2019 to 2021 LTS.unitypackage)

- Builtin : 2019.4.0 or higher


By default, the package is set up for the Universal Render Pipeline (Unity 2022 LTS). In order to use the shaders with the Universal Render Pipeline
you will need to enable the Opaque Texture toggle in the pipeline asset inspector.
(for more information check the documentation linked below). 


Built-in Setup: 
---------
First delete the URP folder and unpack the Builtin-Wet Flood&Rain Puddles Shaders.unitypackage. 


Documentation link : https://docs.google.com/document/d/1SDgEmudN0OSQt-ARpWna50QLQxZ3idX1p2DWUJ-4Ius/edit?usp=sharing

Setp-by-Step Tutorial 
_____________________
Videos
- Wet Shader Flooded Ground - Tutorial : https://youtu.be/Qfv_gW7AKig
- Wet Shader Rainy Puddles - Tutorial    : https://youtu.be/F9U3KB3C8tE


Create a rainy puddle material :
- Right click within a folder in the project window.
- Create/Material
- Select the new material created
- In the inspector window, go to the shader drop-down list
- Select CiconiaStudio/CS_Wet Animation/...
Done

For more information on shader properties, please refer to the documentation linked above.
